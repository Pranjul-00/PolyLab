# PolyLab - Project Management

This file contains foundational mandates for Gemini CLI while working on the PolyLab project.

## Project Vision
PolyLab is an interactive educational platform for visualizing complex chemical concepts. The goal is to evolve it from a static visualizer into a dynamic simulation and experimentation environment.

## Progress Tracking
All project progress is recorded in `PROGRESS.md`. This file should be updated at the end of every significant session or task.

## Standards
- Adhere to React 19 and Vite best practices.
- Maintain the "glassmorphism" UI aesthetic.
- Use `react-force-graph-3d` for complex network visualizations.
- Use `3Dmol.js` for accurate molecular structures.
